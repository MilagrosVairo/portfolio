# 🌟 Portfolio Personal - Milagros Vairo

Bienvenido a mi portfolio personal, un espacio donde comparto mis proyectos de desarrollo y diseño UX/UI.

## ✨ Características

- **Diseño Moderno**: Interfaz limpia y profesional con animaciones sutiles
- **Totalmente Responsive**: Optimizado para todos los dispositivos
- **Modo Zoom en Galería**: Visualización mejorada para imágenes de diseño de gran formato
- **Navegación Fluida**: Scroll suave entre secciones
- **Lightbox Interactivo**: Galería de imágenes para cada proyecto
- **Filtros de Proyecto**: Separación clara entre proyectos de desarrollo y diseño

## 🛠️ Stack Tecnológico

- **React 19** - Biblioteca de interfaz de usuario
- **TypeScript** - Tipado estático para JavaScript
- **Tailwind CSS** - Framework de CSS utility-first
- **Vite** - Build tool ultrarrápido
- **Lucide React** - Biblioteca de iconos moderna

## 🚀 Inicio Rápido

### Requisitos previos

- Node.js (versión 18 o superior)
- npm o yarn

### Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/MilagrosVairo/portfolio.git
cd portfolio
```

2. Instala las dependencias:
```bash
npm install
```

3. Inicia el servidor de desarrollo:
```bash
npm run dev
```

4. Abre tu navegador en `http://localhost:5173`

## 📦 Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye la aplicación para producción
- `npm run preview` - Previsualiza la build de producción
- `npm run lint` - Ejecuta ESLint para verificar la calidad del código
